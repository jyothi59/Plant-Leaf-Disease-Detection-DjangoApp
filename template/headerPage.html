PLANT LEAF DETECTION

by Jyothi

This is my epics project.I created a webiste using django framework, which takes a leaf as an input and it gives back the disease of the leaf.For prediction i used vgg_19 model which is a convolutional neural network with 19 layers deep and network has an input image of size 224X224.Here tomato lead dataset is trained with 10 classes and in each class it has atmost 1400 images.So over all this model trained with 14,000 images.This model has an accuracy of 0.95 and validation accuracy of 0.93.In future i want to extend to other plants to.
